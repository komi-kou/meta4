# UI Design Showcase

5種類のモダンなUIデザインを統合したデモアプリケーション

## 🎨 実装されているUIデモ

### 1. タスク管理（カンバン風）
- ドラッグ＆ドロップ風のタスク管理UI
- To Do / In Progress / Done の3カラム
- タスクの追加、移動、削除機能
- 美しいカード型デザイン

### 2. ポモドーロタイマー
- 円形のビジュアルタイマー
- 作業時間（25分）と休憩時間（5分）の切り替え
- セッション統計の表示
- ミニマルで集中しやすいデザイン

### 3. ダッシュボード
- メトリクスカード（アクティブユーザー、売上など）
- リアルタイム時計ウィジェット
- クイックアクション
- タスクリスト
- 最近のアクティビティ表示

### 4. ノートアプリ
- マークダウン対応のノートエディター
- カテゴリ別フィルタリング（仕事、個人、学習）
- ノートの作成、編集、削除機能
- サイドバー付きの2カラムレイアウト

### 5. アナリティクス（統計表示）
- 週間訪問者数のバーチャート
- デバイス別・トラフィックソース別の統計
- トップページランキング
- インタラクティブなグラフとプログレスバー

## 🚀 技術スタック

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (アニメーション)
- 完全クライアントサイドレンダリング（サーバー不要）

## 📦 デプロイ

### Vercelでデプロイ

```bash
npm run build
# Vercelにデプロイ
```

- 静的サイトとしてエクスポート（`output: 'export'`）
- サーバーサイド機能不要
- ローカルストレージのみ使用

### Renderでデプロイ

1. Static Siteとして設定
2. Build Command: `npm run build`
3. Publish Directory: `out`

## 🛠️ ローカル開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ブラウザでアクセス
# http://localhost:3000
```
